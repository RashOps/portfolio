## 🌟 L'Assistant IA Intégré : ATLAS

La pièce maîtresse de ce portfolio est **Atlas**, un assistant IA intégré de bout en bout (via le **Vercel AI SDK** et **Gemini 2.5 Flash**). 
Atlas possède une base de connaissance (RAG interne) complète du parcours de l'opérateur et peut **déclencher des actions d'interface pures (tools)** pour naviguer programmatiquement l'utilisateur à travers le site.

## 🚀 Architecture des Nœuds (Pages)

*   **[ Home ]** : Dashboard de télémétrie affichant les missions phares (Bento Grid) et la stack globale avec animations séquentioelles.
*   **[ Operator Profile ]** : Identité, ciblage professionnel (Recherche d'alternance) et barres de progression des Data Skills (Science, Engineering, BI).
*   **[ Mission Log ]** : Journal de bord regroupant 6 réalisations majeures (MarketPulse AI, EchoPulse, n8n Workflows) avec tags et filtres.
*   **[ Skill Tree ]** : Arborescence technique structurée de l'ingestion de la donnée jusqu'à sa visualisation (Power BI).
*   **[ Experience Timeline ]** : Parcours dual (Académique & Professionnel) retraçant les pivots (Orange CI, AESIO, Emmaüs).
*   **[ Side Quests ]** : Explorations et projets personnels extérieurs à l'IA pure.
*   **[ Decrypt Demos ]** : Zone de démonstrations interactives incluant une interface technique MarketPulse.

## 🛠 Stack Technique Interactive

- **Framework Front-End :** `Next.js 15+` (App Router) optimisé avec le compilateur React.
- **Moteur IA :** `Vercel AI SDK` couplé au LLM Gemini 1.5/2.5.
- **Styling & UI :** `Tailwind CSS v4` exploitant la puissance du Glassmorphism, de l'opacité et des gradients asymétriques.
- **Animations Scénarisées :** Composants wrappers dynamiques (`FadeIn`, `StaggerContainer`) propulsés par `Framer Motion`.
- **Performance & Tracking :** `Vercel Analytics` et `Speed Insights` intégrés. Serveur local géré par `Bun`.
- **SEO :** Génération dynamique de Vignettes (OpenGraph) en code pur grâce à `@vercel/og` (`Edge Runtime`).

## 💻 Guide d'Installation Rapide (Initialisation)

Pour amorcer la séquence de lancement environnementale locale :

1.  **Récupérer l'archive du pipeline (Clone)**
    ```bash
    git clone https://github.com/RashOps/portfolio.git
    cd portfolio
    ```

2.  **Installer l'environnement de build (Bun recommandé)**
    ```bash
    bun install
    ```

3.  **Configurer les clés cryptographiques (Variables d'environnement)**
    Créez un fichier `.env.local` et ajoutez votre clé pour initialiser ATLAS :
    ```env
    GOOGLE_GENERATIVE_AI_API_KEY=votre_cle_gemini_ici
    ```

4.  **Lancer le flux serveur**
    ```bash
    bun run dev
    ```
